# Contributing / 贡献指南

Thank you for considering contributing! 🎉

## 中文

### 🐛 报告 Bug
- 在 GitHub Issues 中创建新 Issue
- 使用 Bug Report 模板
- 请附上：macOS 版本、Python 版本、错误日志、复现步骤

### 💡 功能建议
- 在 Discussions 或 Issues 中提出
- 说明使用场景和预期行为

### 🔧 提交 PR

1. Fork 本仓库
2. 创建新分支: `git checkout -b feat/your-feature`
3. 提交更改: `git commit -m "feat: add your feature"`
4. 推送到分支: `git push origin feat/your-feature`
5. 创建 Pull Request

### ✅ PR 检查清单
- [ ] 代码兼容 Python 3.9+
- [ ] 测试通过
- [ ] README 已更新（如需要）
- [ ] 中英文双语注释

## English

### 🐛 Report a Bug
- Open a GitHub Issue using the Bug Report template
- Include: macOS version, Python version, error logs, reproduction steps

### 💡 Feature Request
- Open a Discussion or Issue
- Describe the use case and expected behavior

### 🔧 Submit a PR

1. Fork the repo
2. Create a branch: `git checkout -b feat/your-feature`
3. Commit: `git commit -m "feat: add your feature"`
4. Push: `git push origin feat/your-feature`
5. Open a Pull Request

### ✅ PR Checklist
- [ ] Code compatible with Python 3.9+
- [ ] Tests pass
- [ ] README updated (if needed)
- [ ] Bilingual comments (Chinese + English)
